# Interactive Story

This is a simple interactive story project using HTML, CSS, and JavaScript.

## How it works
- Open `index.html` in a browser.
- Click the "Next" button to move through the story.
- The story progresses in parts and ends with "The End."

## Files included
- `index.html` — main webpage
- `style.css` — styling
- `script.js` — JavaScript for interaction
- `story.txt` — story content
- `README.md` — project info
- `.gitignore` — ignored files
